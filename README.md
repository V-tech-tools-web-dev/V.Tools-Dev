# Usage
<!-- usage -->
```sh-session
$ npm install -g v_tool
$ v_tool COMMAND
running command...
$ v_tool (-v|--version|version)
v_tool/1.0.1 win32-x64 node-v14.15.4
$ v_tool --help [COMMAND]
USAGE
  $ v_tool COMMAND
...
```
